# OLED Games

Small PlatformIO project for an OLED-based game. The main source file is `src/main.cpp`.

Build and upload (PlatformIO):

```bash
platformio run --environment <env> --target upload
```

Open the project in VS Code and use the PlatformIO extension to select the correct environment and serial monitor.
